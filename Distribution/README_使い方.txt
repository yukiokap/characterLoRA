# Character LoRA Manager ユーザーガイド

この度は「Character LoRA Manager」をご利用いただきありがとうございます！
このアプリは、Stable Diffusionなどの画像生成AIで使う「キャラクター設定」や「LoRAモデル」を、直感的かつ強力に管理するためのツールです。

---

## 🚀 1. はじめてみよう（起動方法）

特別なインストール作業は一切不要です。

1. **ファイルを展開**: 配布された Zip ファイルを、デスクトップなど好きな場所に解凍（展開）してください。
2. **アプリを起動**: フォルダ内にある **`AiManager.exe`** をダブルクリックしてください。
3. **ブラウザを確認**: 数秒後に自動的にブラウザが立ち上がり、管理画面が表示されます。
   ※自動で開かない場合は、ブラウザで `http://localhost:3001` にアクセスしてください。

### 🛡 起動時に警告が出た場合
Windowsの「PCが保護されました」という画面が出た場合は、「詳細情報」をクリックし、その後に表示される「実行」ボタンを押してください。

---

## ⚙️ 2. 最初に行う設定

画面右上の **Settings（歯車アイコン）** をクリックして、以下の設定を行ってください。

*   **LoRA Directory**: あなたが LoRA ファイル（.safetensors）を保存しているフォルダパスを入力します。
*   **Wildcards Directory**: ワイルドカード（.txt）を保存している共有フォルダなどがあればそのパスを入力します。
*   **Gemini API Key**: キャラクターの衣装分析機能（AI機能）を使いたい場合は、GoogleのGemini APIキーを入力してください。

---

## 📂 3. データはどこに保存されるの？

このアプリは「ポータブル仕様」です。EXEファイルがある場所の周辺にすべてのデータが溜まっていきます。

*   **`data/` フォルダ**: 
    登録したキャラクター情報、LoRAのメモ、AIの分析結果、環境設定などがすべてここに入ります。
*   **`uploads/` フォルダ**: 
    キャラクター一覧に表示するためにあなたがアップロードした画像がここに入ります。

**💡 バックアップの方法**:
新しいPCに乗り換えるときは、この「CharacterLoraManager」フォルダごとコピーすれば、すべての設定やキャラクターリストをそのまま引き継ぐことができます。

---

## ❓ 4. よくある質問とトラブル解決

**Q. 黒い画面（コンソール）が出てくるけど閉じていい？**
A. アプリが動いている間は開いたままにしておいてください。最小化しておくのがおすすめです。使い終わったら、その画面を「×」で閉じればアプリも終了します。

**Q. ブラウザに「ページが表示できません」と出る**
A. 他のアプリが「3001番ポート」を使っている可能性があります。また、一度アプリを閉じて、タスクマネージャーで「AiManager.exe」が残っていないか確認してから再起動してみてください。

**Q. キャラクターの順番が変わらない**
A. 並び替え（ドラッグ＆ドロップ）をしたあとは、自動でデータフォルダ内の `characters.json` に保存されます。

## 📝 免責事項・その他
このアプリはローカル環境で動作する個人向けツールです。
あなたの LoRA ファイルそのものを書き換えたり削除したりすることはありませんが、大切なデータは念のためバックアップを取っておくことを推奨します。
